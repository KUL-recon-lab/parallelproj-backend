Backprojection using the Joseph 3D algorithm.

Note
All pointers can be host pointers, CUDA device pointers, or CUDA managed pointers.
Parameters
xstartPointer to array of shape [3*nlors] with the coordinates of the start points of the LORs. The start coordinates of the n-th LOR are at xstart[n*3 + i] with i = 0,1,2. Units are the ones of voxsize.
xendPointer to array of shape [3*nlors] with the coordinates of the end points of the LORs. The end coordinates of the n-th LOR are at xend[n*3 + i] with i = 0,1,2. Units are the ones of voxsize.
imgPointer to array of shape [n0*n1*n2] containing the 3D image used for backprojection (output). The pixel [i,j,k] is stored at [n1*n2*i + n2*j + k]. !! Values are added to the existing array !!
img_originPointer to array [x0_0, x0_1, x0_2] of coordinates of the center of the [0,0,0] voxel.
voxsizePointer to array [vs0, vs1, vs2] of the voxel sizes.
pPointer to array of length nlors with the values to be backprojected.
nvoxelsNumber of voxels in the image.
nlorsNumber of geometrical LORs.
img_dimPointer to array with dimensions of the image [n0, n1, n2].
device_idID of the device to use for computation (default: 0).
threadsperblockNumber of threads per block for GPU computation (default: 64).

Note
All pointers can be host pointers, CUDA device pointers, or CUDA managed pointers.
Parameters
